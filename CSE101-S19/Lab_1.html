Lab 1: Programming Environment

Part 1: Python and PyCharm installation

In this first part of the lab, the goal is to get Python and an IDE (PyCharm) installed on your computer. If you have a laptop, bring it to the lab. We suggest that you work in pairs with someone who has a laptop with the same operating system (Windows or Mac or Linux) as you do.

Installation instructions for Python and PyCharm are provided as part of Lecture 2. Please follow the instructions carefully and exactly as deviating from them can cause unexpected errors.

Make sure that you can create a project with a simple Python program (e.g., hello.py) and run it successfully.

Demonstrate to the TA that your Python installation and PyCharm works using the following steps:

1)     Execute the following lines from the Python prompt:

'''Illustrate input and print.'''

 

>>> applicant = input("Enter the applicant's name: ")

>>> interviewer = input("Enter the interviewer's name: ")

>>> time = input("Enter the appointment time: ")

>>> print(interviewer, "will interview", applicant, "at", time)

2)     Create a Python file named interview.py using PyCharm. Add above lines in created file. Save and run the program using PyCharm.
